# gae-go-example02
Hello world app on AppEngine Go / Deploy with AppEngine Admin API

## Prerequisites

- Go 1.11.x
- Set environment variable `GO111MODULE=on`

## Build / Run

Run local web server without build:

```shell
go run server.go
```

or, build and run:

```shell
go build
./gae-go-example02
```

## Test

```
go test
```
